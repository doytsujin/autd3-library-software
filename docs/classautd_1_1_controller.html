pure virtual

Open device by link type and location.

Parameters
[in]typeLinkType
[in]locationThe scheme of location is as follows:
  1. ETHERCAT, TwinCAT: <ams net id> or <ipv4 addr>:<ams net id> (ex. 192.168.1.2:192.168.1.3.1.1 ). The ipv4 addr will be extracted from leading 4 octets of ams net id if not specified. If empty, localhost
  2. SOEM: Network interface name (can be obtained by EnumerateAdapters())
  3. EMULATOR: <ipv4 addr>:<port> of the computer running emulator

virtual void autd::Controller::StartSTModulation ( double  freq)
pure virtual

Start Spatio-Temporal Modulation.

Parameters
[in]freqFrequency of STM modulation

Generate STM modulation by switching gains appended by AppendSTMGain() at the freq. The accuracy depends on the computer, for example, about 1ms on Windows. Note that it is affected by interruptions, and so on.
